需要安装

terminal-notifier

增强通知 — 有声音、可自定义图标

1
安装
brew install terminal-notifier
2
配置命令
terminal-notifier -title 'Claude Code' -message 'Claude Code needs your attention' -sound default -contentImage ~/.claude/assets/claude-icon.png

优点

有声音提醒，不容易错过
右侧可展示自定义图标
支持点击跳转（-open 参数）

缺点

需要 brew 安装
左侧图标无法自定义（macOS 限制）
通知仍会自动消失
