Windows 通知配置

配置结构一样，只需替换 command 部分

1
msg 命令 最简单
零依赖，一行搞定，弹出系统消息框
msg %username% "Claude Code 通知"
2
PowerShell MessageBox ⭐ 推荐
无需额外文件，直接调用 .NET 弹窗，带图标
powershell.exe -Command "Add-Type -AssemblyName System.Windows.Forms; [System.Windows.Forms.MessageBox]::Show('Claude Code 通知', 'Notification', 'OK', 'Information')"
3
PowerShell Toast 通知
系统通知横幅样式，类似 macOS 的 display notification
powershell.exe -ExecutionPolicy Bypass -File C:/Users/你的用户名/AppData/Local/Temp/claude_toast_notify.ps1 -Title 'Claude Code' -Message 'Claude Code needs your attention'
