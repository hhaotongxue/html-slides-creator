Toast 脚本

方案三需要先创建一个 .ps1 脚本文件

C:/Users/你的用户名/AppData/Local/Temp/claude_toast_notify.ps1
param( [string]$Title = "Claude Code", [string]$Message = "通知" ) # 加载 Windows Runtime 类型 [Windows.UI.Notifications.ToastNotificationManager, Windows.UI.Notifications, ContentType = WindowsRuntime] | Out-Null $template = "<toast><visual>...</visual></toast>" $xml = New-Object Windows.Data.Xml.Dom.XmlDocument $xml.LoadXml($template) $toast = [Windows.UI.Notifications .ToastNotification]::new($xml) [Windows.UI.Notifications .ToastNotificationManager] ::CreateToastNotifier("Claude Code") .Show($toast)

按 matcher 分别通知

permission_prompt → 需要权限批准
idle_prompt → Claude 任务完成
auth_success → 认证成功
elicitation_dialog → 有新的对话
配置多条规则，不同事件显示不同消息，一目了然
