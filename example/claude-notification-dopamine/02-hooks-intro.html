Hooks
前置准备

Claude Code 支持通过 hooks 配置 Notification 事件，当 AI 需要你注意时自动触发命令。

配置文件路径：~/.claude/settings.json

matcher 通知类型

permission_prompt 需要权限批准时
idle_prompt 空闲等待输入时
auth_success 认证成功时
elicitation_dialog 弹出对话框时
"" 留空 = 所有类型都触发
// ~/.claude/settings.json { "hooks": { "Notification": [ { "matcher": "", "hooks": [ { "type": "command", "command": "你的通知命令" } ] } ] } }
