## [0.1.0] - 2026-03-18

### Added
- Initial release of HTML Slides Creator
- Three visual styles: Dopamine, Jelly, and Macaron
- 16:9 canvas with automatic scaling
- Independent HTML files for each slide
- Built-in navigation and progress bar
- Support for multiple layout modes
- Scale.js for responsive canvas adaptation
